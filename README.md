## Tools (8)

| Tool | Description |
|------|-------------|
| `job_search` | Search jobs by title, skill, location. Filter by type and recency. |
| `salary_insights` | Salary ranges for a role in a location. |
| `remote_jobs` | Find remote-only positions for a role or skill. |
| `company_jobs` | All open positions at a specific company. |
| `job_trends` | Trending job titles and skills in demand for a field. Rising searches + interest |
| `job_compare` | Compare job markets across cities for a role: job count, salaries, remote option |
| `job_alerts_check` | How many new jobs posted today/3 days/week for a query. Quick market pulse. |
| `health_check` | Server status. |
